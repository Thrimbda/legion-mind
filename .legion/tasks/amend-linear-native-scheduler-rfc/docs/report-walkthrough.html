Linear Native Agent RFC amendment

本交付把 review blocking points 纳入 Linear + Legion scheduler 设计。重点是 native control plane、成功终态门禁、lifecycle gate、snapshot revalidation 和 8 个 WI 的一致性。

Review RFC: PASS Verification: PASS Review Change: PASS Runtime code: none

What changed

Native Agent layer

Issue.delegate, AgentSession, Activities, Plan, externalUrls and stop signal are now presentation/control plane. Scheduler DB remains machine truth.

Terminal semantics

Done means run_terminal_success. Canceled, abandoned and closed-unmerged are terminal non-success and do not unlock downstream by default.

Lifecycle gate

PR merged is not enough. Success also requires evidence PASS plus git-worktree-pr cleanup and main refresh. Missing lifecycle evidence becomes lifecycle_blocked.

WI alignment

All 8 WI docs were updated. WI-06 now waits for WI-05 so parallel lock release cannot precede PR terminal tracking.
