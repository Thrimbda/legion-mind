Reviewer Summary

lgmind now has a product-level setup flow with --agent opencode|openclaw, while setup-opencode remains the direct OpenCode alias. Text output is quiet by default; --verbose and --json retain detailed output.

Scope

In scope
  • Aggregator entrypoint and runtime selection.
  • Quiet default reporter behavior.
  • OpenClaw package surface and docs/tests.
Out of scope
  • No per-agent subset selection.
  • No new runtimes.
  • No npm publish.

What Changed / What Was Decided

  • lgmind routes setup/lifecycle commands to OpenCode or OpenClaw.
  • setup maps to runtime install and can prompt only in TTY.
  • --agent is the documented runtime selector; --runtime is an alias.
  • Default text mode hides OK event noise, but keeps warnings/errors and final summaries.

Risks and Limits
